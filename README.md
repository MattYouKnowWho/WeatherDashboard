# WeatherDashboard
Using OpenWeather API to display a 5-day forecast and have history for each city the user searched. Of course, making it visually pleasing as well.

# Technology Used 

- Javascript
- CSS
- HMTL
- Bootstrap
- Node
- OpenWeatherMap API
